1. Key Components of a Computer: Central Processing Unit (CPU): Often referred to as the computer's "brain," the CPU carries out program instructions, performs calculations, and manages data. It consists of two essential components: the arithmetic logic unit (ALU) for calculations and the control unit for executing instructions. Memory (RAM): Random Access Memory (RAM) serves as a computer's short-term memory. It temporarily stores actively used data for quick access and manipulation. Storage Devices: Computers come equipped with both primary storage (such as Solid State Drives - SSDs and Hard Disk Drives - HDDs) and secondary storage (like external drives and USB drives) for long-term data retention. These devices house the operating system, software, and user data. Motherboard: The motherboard is the primary circuit board within the computer. It connects and facilitates communication among all essential components, including the CPU, RAM, storage devices, and peripherals.
